# MiniProgHeaderView
>最近项目需要，仿了一下微信小程序下拉小程序入口

### 实现的功能
1. 在tableView下拉显示小程序入口
2. 高仿实现三个点的动画特效
3. 下拉过程震动提示
4. 小程序入口界面支持动态高度变化

### 使用到的第三方框架

* https://github.com/Yalantis/Persei
* https://github.com/CoderMJLee/MJRefresh
